How does it work?

Schedule Parser is an amateur project carried out by Lei Yang.

I used to add class information to calendar app manually, but it is a boring job and takes much time. So I wrote a program to automate the process. At first all I want is a simple Python script taking in an HTML file downloaded from elective.pku.edu.cn and generate an .ics file containing class information so that I can add information to calendar app easily. As I found it useful to many people, I decided to release it. However, it is impossible to make every user install a correct version of Python interpreter and all dependencies, so the idea of making a web app doing the same job came to my mind.

Schedule Parser takes in your Peking University account information, logs in to elective.pku.edu.cn and downloads the course election results. Then it parses the file and provides you an .ics file which is supported by Apple Calendar, Google Calendar, Microsoft Outlook and many other calendar apps.

It is a open-source app and I host the code on GitHub.
